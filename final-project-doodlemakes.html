Desciption of Doodle Makes

This application titled, Doodle Makes, will be an html webpage displaying the various forms of artworks that I have created. This webpage will display multiple pictures of paintings, sketches, and even doodles from various points in my life. Each painting, doodle, and sketch will have a title as well as a brief description of the painting itself. The viewer will be able to look through the artworks easily and effectively.

Prioritized Desired Features

  1. Colorful and Appealing background
  2. Different labelled sections for different forms of art
  3. Artistic background that the users can view (essentially a personal artistic background)

Unprioritized Desired Features

  1. Ability for users to comment on the webpage under different artworks
  2. Multiple pages that the user can click onto
  3. Ability for users to make requests on artworks they'd like to see done

Languages and Platforms

I am going to be using html, CSS and possibly GitHub/Azure to implement my project. When it comes to technical features I will be using the information we learned through the W3 website and teh information from Sprint 3 when we created our getting to know eachother webpages. I will make the webpage as aesthetically pleasing and easy to view as possible. When it comes to features that go beyond what we learned in class, that would include making a creative background that isn't just a simple color, utilizing different styles and fonts that go with the aesthitic I am trying to create, and creating a comments section on the webpage.
